Настройки профиля

Избранные репозитории отмечаются ⭐ во вкладке «Репозитории». Они будут описаны максимально подробно и идти первыми в профильном README.

Действия

«Собрать кэш» — клонирует все репозитории и генерирует описания (без пуша).
«Предпросмотр» — показывает сгенерированный README из кэша.
«Сгенерировать и запушить» — полный цикл: кэш → README → пуш в профильный репо.
